# Simple exemple hello-world app with vuejs and lando

## Project setup
```
lando start
```

### Compiles and hot-reloads for development
```
lando dev
```

### Compiles and minifies for production
```
lando yarn build
```

### Lints and fixes files
```
lando yarn lint
```
